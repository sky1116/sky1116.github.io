重構

Model

ViewModel

MainViewModel.java

item layout

接下來，先把 item 的 layout 改成 data binding 的格式。
user_list_item.xml

Adapter

Adpater 要處理的部份比較多，這裡會拆幾個部分來說明。
先就 ViewHolder 的部份來看，ViewHolder 代表著每一個 item 的 layout，相對的，也對應著一個 UserInfo，建構式改為傳入 UserListItemBinding，新增 bind function 用來綁定 UserInfo，然後直接呼叫 executependingbindings() 更新畫面。

onCreateViewHolder 使用 inflate 的方式來產生 UserListItemBinding。

onBindViewHolder 移除 findViewById，使用 bind 來綁定資料更新畫面。

最後新增一個 refresh 的方法，搭配 DiffUtil 來做到 RecyclerView 的差異更新，這裡先不考慮大量資料使用背景執行緒做比對，之後再針對這個做優化。
