Kotlin 學習筆記 (一)

Posted on 2020-11-01 | Post modified: 2020-12-20 | In android

這個系列是我在學習 Kotlin 過程中所做的筆記，如果沒學過的可以順著筆記一起來了解 Kotlin，有學過的也可以藉此來複習一下吧

常數及變數

val 常數宣告，相當於 Java 的 final，值不可再更改
var 變數宣告，可重新賦值

基本型別

基本型別皆為首字大寫

型別 位元
Long 64
Int 32
Short 16
Byte 8
Double 64
Float 32

型別推斷

Kotlin 具有型別推斷的能力，宣告變數時可不寫類別，也可使用：將類別加在變數後

型別推斷決定類別後，不可再指定不同類別的值 (與 javascript 不同)

字串輸出佔位符

在字串中要輸出變數或運算結果時可用 ${}

Tuple (元組)

在 Kotlin 中有 Pair 跟 Triple

Pair 可存兩個值

Triple 可存三個值

布林值

布林值的類別為 Boolean，只有兩個值 true 和 false

例外處理

參考相同或結構相同

比對參考是否相同使用 === 或是 !==
比對結構是否相同使用 == 或是 !=
