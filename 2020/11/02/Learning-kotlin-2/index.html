區間運算子 (Range)

在 Kotlin 中，.. 運算子可以用來表示兩個值的區間，包含起始值與終止值，並使用 in 來判斷一個值是否在區間內

結合 for 迴圈有循環的功能

downTo

使用 downTo() 可創建倒序排序的區間，需將終止值當引數帶入

rangeTo

使用 rangeTo() 可創建到指定終止值的正序區間，需將終止值當引數帶入

step

區間創建後，亦可更改區間產生新的區間，可以使用 step() 更改區間的間隔，需將間隔當引數帶入

reserved

因為無法使用負數產生遞減的區間，可以使用 reversed() 來反轉區間

when

帶參數的 when

when 可以說是 Java 的 switch 升級版

  • 條件分支最後都要有 else，相當於 Java 的 default
  • 編譯器可自動推斷出 when 所有分支情況時可省略 else，通常是列舉或內部類別

when 與 if-else, try-catch 一樣可做為表達式，每個分支結果都是回傳值

如果分支處理流程相同，可分併分支用逗號隔開

when 語句同表達式，所以可省略變數

類型層級

可參考這篇

Any

Kotlin 中最上層的類型是 Any，為任意類型，相當於 Java 中的 Object

Unit

Unit 等同於 Java 中的 void，只是 void 不是一個類型，而 Unit 是類型，用於實現一個單例，當一個函數定義回傳一個 Unit 時，會回傳一個 Unit 的單例

Nothing

跟 Any 相反，Nothing 是所有類型的子類別，是一種空類型 (uninhabited type)，throw 的表達式就是回傳 Nothing，用來標記永遠不會到達的代碼位置，或是標記一個永遠不會回傳值的函數，當調用後編譯器就不再繼續執行

如果用 null 來初始化一個要推斷的值，當沒有其他可確認類型的訊息時，編譯器會推斷出 Nothing?
